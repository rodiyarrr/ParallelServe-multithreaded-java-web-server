# Multithreaded Web Server in Java

A high-performance multithreaded web server built using Java, capable of handling multiple client requests concurrently using socket programming and threads.

---

## 🚀 Features

- Handles multiple client requests simultaneously
- Implements HTTP request-response cycle
- Supports basic HTTP methods (GET)
- Serves static files (HTML, CSS)
- Multithreading for improved performance
- Error handling (404 Not Found)

---

## 🛠️ Tech Stack

- Java (Core Java)
- Socket Programming
- Multithreading
- HTTP Protocol
